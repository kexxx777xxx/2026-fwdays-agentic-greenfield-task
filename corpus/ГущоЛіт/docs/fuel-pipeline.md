# ⛽ Паливна система: від гущі до польоту

Це ядро всього проєкту. Тут спита кава перетворюється на пальне.
Найкрасивіше те, що **хімія справжня** — на спитій гущі дійсно роблять
біодизель. CREMA-1 просто возить цей завод із собою.

---

## Чому саме гуща?

Спита кавова гуща містить **15–20% олії**. Ця олія за складом близька до
рослинних олій, з яких роблять біодизель. А ще гущі — гори: одна кав'ярня
дає десятки кілограмів на день і платить за те, щоб її вивезли.

**Ми перетворили статтю витрат міста на джерело пального.**

---

## Конвеєр (5 кроків)

```
[1] ЗБІР      [2] СУШКА     [3] ВІДЖИМ    [4] РЕАКЦІЯ      [5] ПОЛІТ
спита гуща →  −вода     →   +олія     →  трансетерифік. → біодизель B100
60% вологи    суха       ~18% маси      олія+метанол       → турбіна
```

### 1. Збір
Гуща з кав'ярень злітається в герметичний бункер. Сортування не треба —
підходить будь-яка: арабіка, робуста, навіть учорашня «друга чашка».

### 2. Сушка
Гуща мокра. Сушимо теплом, яке турбіна все одно викидає в атмосферу.
Ціль — менше 10% вологи, інакше реакція не піде.

### 3. Віджим олії
Гвинтовий прес вичавлює кавову олію. Залишок (макуха) не викидається —
з нього пресуються **паливні брикети** й добриво.

### 4. Трансетерифікація
Серце процесу. Спрощено:

```
кавова олія + метанол --(каталізатор KOH)--> біодизель (B100) + гліцерин
```

Каталізатор пришвидшує реакцію, гліцерин осідає вниз і зливається.
Зверху — чистий біодизель, готовий горіти.

### 5. Політ
B100 іде в мікротурбіну → електрика → вентилятори. Готово, ви летите
на тому, що випили зранку.

## Баланс мас (приблизно)

| Вхід / вихід | Кількість |
|---|---|
| Спита гуща (суха) | 100 кг |
| → кавова олія | ~18 кг |
| → біодизель B100 | ~16 л |
| → дальність польоту | ~90 км |
| Побічно: брикети | ~70 кг |
| Побічно: гліцерин | ~2 л |

**Правило великого пальця:** 50 кг сухої гущі ≈ 45 км польоту.

## Побічні продукти (нічого не пропадає)

- 🔥 **Брикети** з макухи — паливо для наземної сушки або продаж.
- 🌱 **Добриво** — азот і калій для садів.
- 💧 **Гліцерин** — у мило чи косметику.

CREMA-1 не лишає сміття. Вона **їсть** сміття.
