# 🧩 Архітектура системи CREMA-1

Документ описує, **з чого складається** ГущоЛіт і як енергія тече від
сміттєвого бака кав'ярні до висоти. Це базис для відтворення проєкту.

---

## Загальна схема v1

```
        ┌──────────────────────────────────────────────┐
        │                  CREMA-1                       │
        │                                                │
   ☕   │  [1] Бункер гущі ──► [2] Сушка/Прес ──► олія   │
  ──────┼─►                                       │      │
        │                                          ▼     │
        │                        [3] Реактор біодизелю   │
        │                                          │     │
        │                                          ▼     │
        │   [6] Авіоніка ◄── [4] Мікротурбіна-генератор  │
        │        │                                 │     │
        │        ▼                                 ▼     │
        │   [5] Батарея-буфер ──► 4× EDF ──► 🌀 ПІДЙОМ   │
        └──────────────────────────────────────────────┘
```

Логіка проста: **кава робить струм, струм крутить вентилятори.**
Турбіна не штовхає машину напряму — вона лише заряджає батарею.
Це робить політ тихим, плавним і керованим.

## Підсистеми

### [1] Бункер гущі
Герметичний бункер на 60 кг спитої гущі. Має шнек, що подає сировину
далі дозовано. Заправляється згори, як паливний бак, але лопаткою.

### [2] Сушка та прес
Гуща приходить мокрою (≈ 60% вологи). Вузол сушить її теплом від турбіни
(нічого не пропадає) і віджимає олію гвинтовим пресом.

### [3] Реактор біодизелю
Маленький бортовий хімреактор. Олія + метанол + каталізатор →
біодизель B100. Деталі — у [паливній системі](fuel-pipeline.md).

### [4] Мікротурбіна-генератор
Серце машини. Спалює біодизель, обертає генератор. Видає сталі ~80 кВт
електрики. Працює на одних обертах → довговічна й передбачувана.

### [5] Батарея-буфер + 4× EDF
Літій-буфер згладжує піки. Чотири електричні канальні вентилятори
(electric ducted fans) у поворотних гондолах дають вертикальний зліт
і горизонтальний політ.

### [6] Авіоніка
Політний контролер балансує тягу 4 вентиляторів, керує гондолами,
стежить за рівнем гущі та температурою турбіни. Має «режими кави»
(див. [довідник аеро-баристи](../docs2/barista-handbook.md)).

## Компоненти

| № | Вузол | Функція | Критичність |
|---|---|---|---|
| 1 | Бункер + шнек | зберігання й подача гущі | 🟡 середня |
| 2 | Сушка-прес | волога → олія | 🔴 висока |
| 3 | Реактор B100 | олія → біодизель | 🔴 висока |
| 4 | Мікротурбіна | біодизель → струм | 🔴 висока |
| 5 | Батарея + EDF | струм → тяга | 🔴 висока |
| 6 | Авіоніка | керування й безпека | 🔴 висока |
| 7 | Рама (алюміній+карбон) | каркас | 🟡 середня |

## Потік енергії

```
гуща (хім. енергія) → олія → біодизель → тепло+рух турбіни
   → електрика → батарея → вентилятори → кінетична енергія → ВИСОТА
```

ККД ланцюга невеликий, але **сировина безкоштовна та нескінченна**,
тож економіка тримається не на ефективності, а на тому, що паливо —
це чужий непотріб.
